# spring-react-app
A Demo Todo list created with React and Spring boot.

### First build the backend
 ```
 gradle clean build
 ```
 
 ### Build the Frontend
 ```
gradle deployFrontend
 ```
 
  ### Run Spring boot
 ```
gradle bootRun
 ```
 
 ### then go to the url:
  ```
 http://localhost:8080/
  ```
